## Regenerating Documentation

The API documentation is auto-generated from TypeScript source using [TypeDoc](https://typedoc.org/) with the [typedoc-plugin-markdown](https://github.com/tgreyuk/typedoc-plugin-markdown) plugin.

To regenerate the documentation locally:

```bash
npm run docs
```

This will:
1. Clean the `docs/reference/ts/_generated` directory
2. Run TypeDoc to generate fresh Markdown files from `src/public-api.ts`

## Note on Generated Files

The `_generated/` directory contains auto-generated Markdown files. These files are excluded from version control by default (see `.gitignore`). If you need to commit the generated docs for static hosting, remove the corresponding line from `.gitignore`.
